## How to Run

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd HelloMaven
```

Compile

```bash
mvn compile
```

Run

```bash
mvn exec:java "-Dexec.mainClass=com.iyoob.lab01.HelloMaven"
```
